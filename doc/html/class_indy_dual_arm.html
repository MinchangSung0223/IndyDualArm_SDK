Detailed Description

12-DoF DualArm 동역학/기구학

주요 기능

  • Forward / Inverse Dynamics
  • Forward Kinematics (POE)
  • Joint/Task-space trajectory generator
  • H∞ / QP 기반 컨트롤러
  • slx폴더의 simulink 파일들을 기반으로 c code generation. 사용 예시는 examples/test_dualarm.cpp 참고.
